    TripWhat

    by Parsing Nemo

    TripWhat is an AI-powered travel planner that creates smart, personalized itineraries using intelligent agents with MCP + LangGraph. It understands your intent using LLM-based detection, then uses 10+ integrated tools like web search, routing, and restaurant discovery. Get real-time travel insights, multi-stop route planning, and map-based trip visualization in a smooth chat experience.

    React + TypeScript Node.js + Express MongoDB LangGraph (MCP) OpenAI GPT-4o-mini
    GitHub

    Async Synapse

    by SyncSparks

    SyncSparks is an AI-powered collaboration platform for creative teams to transform ideas into scripts, scenes, and simulations. It provides role-based dashboards, real-time editing with CRDT, and secure squad workflows. Powered by Gemini 2.0 Flash + LangChain, it boosts creativity with AI-assisted writing, summarization, and optimization.

    Next.js + TypeScript Node.js + Express MongoDB Gemini 2.0 Flash LangChain WebSockets + CRDT
    GitHub
    2nd Place

    MediDose

    by Pookie Blinders

    MediDose is a MERN-based medication adherence platform built to help users manage medicines and follow their dose schedules properly. It supports smart medication tracking, CRUD-based dose management, and multi-channel reminders via Email, SMS, and WhatsApp. The system also triggers SOS alerts for prolonged missed doses and provides analytics for adherence monitoring. With Google Calendar sync and an AI-powered assistant (Cura Bot), it makes medication planning smarter and more reliable.

    MongoDB Express.js React Node.js Groq NLP / AI Bot
    Demo

    Object Detection

    by Tricksters

    A computer vision project focused on pothole/crater detection in road images using YOLO-based object detection. The goal is to build a fast and accurate model that identifies potholes and draws bounding boxes around them. A key challenge is ensuring strong performance in difficult real-world conditions like night lighting, glare, shadows, and occlusions.

    Python YOLOv8 Deep Learning

    About CodeSangam 2025

    CodeSangam 2025 is a month-long development fest organized by the Computer Coding Club (CCC), MNNIT Allahabad, where innovation meets execution. It brings together developers, designers, and tech enthusiasts to build real-world projects, learn modern technologies, and compete across multiple development domains.

    The fest is designed to promote hands-on building culture, teamwork, and skill development through a series of exciting events conducted over multiple weeks.

    What Makes CodeSangam Special

    • Multiple events covering diverse development fields
    • Hands-on project building with real-world use cases
    • Competitive environment with exciting prizes and recognition
    • Learning-focused challenges to improve development skills
    • Collaboration and networking with talented peers at MNNIT

    Events / Tracks

    This year, CodeSangam featured 5 major development events:

    1. Webster - Web development event
    2. Droidrush - Android app development event
    3. Logical Rhythm - ML-based development event
    4. Softablitz - Java-based desktop app development event
    5. ContriHub - Open-source contribution challenge
