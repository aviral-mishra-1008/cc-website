    Sign It or Miss It

    by 4Sum

    Sign It or Miss It is an interactive Unity game that use computer vision and hand tracking to provide an immersive experience. The game uses OpenCV and MediaPipe to track hand movements and generate a corresponding 3D hand in Unity. Players must pass their virtual hand through walls with cutout shapes to progress through the game.

    OpenCV MediaPipe cvzone C#

    About Weekend of Code

    Weekend of Code is a multi-event coding fest exclusively for first- and second-year students, bringing together three core competitions—DevJam (development hackathon), Turing’s Playground (AI/ML hackathon), and CodeStart (ICPC-style coding contest). It’s a focused platform to learn, build, compete, and push your problem-solving skills.

    What Makes Weekend of Code Special

    • Three Competitive Events across development, AI/ML, and competitive programming
    • Industry Mentorship from leading tech companies
    • Exclusively for 1st & 2nd Years
    • Mentorship & Guidance from peers
    • Workshops and learning sessions
