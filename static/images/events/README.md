# Events Images for Landing Page Carousel

This folder contains images for the events carousel on the landing page.

## Required Images:

Add the following images to this folder:

1. **winter-hackathon-2025.jpg** (800x600px recommended)
   - Image showcasing the Winter Hackathon event
   - Should show coding, teamwork, or hackathon atmosphere

2. **docker-workshop.jpg** (800x600px recommended)
   - Docker/Kubernetes themed image
   - Tech workshop or containers visualization

3. **summer-hackathon-2025.jpg** (800x600px recommended)
   - Past hackathon event photo
   - Teams working, presentations, or winners

4. **contrihub-2025.jpg** (800x600px recommended)
   - Open source/collaboration themed
   - Code contributions, GitHub, or community

5. **ai-ml-workshop.jpg** (800x600px recommended)
   - AI/ML themed image
   - Neural networks, data science, or ML models

## Image Guidelines:

- **Format**: JPG or PNG
- **Size**: 800x600px (4:3 ratio) - will be displayed at 320px height
- **Quality**: High quality, web-optimized (under 200KB per image)
- **Theme**: Should match CC Club's dark burgundy & gold aesthetic
- **Content**: Clear, professional, relevant to the event

## Placeholder Sources:

If you don't have custom images yet, you can use:
- Unsplash (https://unsplash.com/) - Search: "hackathon", "coding", "workshop", "teamwork"
- Pexels (https://pexels.com/) - Free stock photos
- Your own event photos from past CC Club events
