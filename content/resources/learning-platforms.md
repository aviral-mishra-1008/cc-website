# Learning Platforms

### Comprehensive Courses

**FreeCodeCamp** - [freecodecamp.org](https://www.freecodecamp.org/)  
Free, comprehensive web development curriculum with certificates.  
*Best for: Beginners, Web Development*

**The Odin Project** - [theodinproject.com](https://www.theodinproject.com/)  
Full-stack JavaScript or Ruby on Rails paths.  
*Best for: Self-paced learners, Full Stack*

**FullStackOpen** - [fullstackopen.com](https://fullstackopen.com/)  
University of Helsinki's modern web development course.  
*Best for: Intermediate learners, React/Node.js*

**CS50** - [cs50.harvard.edu](https://cs50.harvard.edu/)  
Harvard's introduction to computer science.  
*Best for: CS fundamentals*
